# fx_rl

FX trading systems with deep reinforcement learning. 

## Getting Started

### Installing
Installing past data.
```
git clone https://github.com/utarumo/fx_rl.git
cd fx_rl
wget http://www.forextester.com/templates/data/files/USDJPY.zip
unzip USDJPY.zip
```

## Runnning
```
python dqn_fx.py
```
